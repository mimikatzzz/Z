How Did a Website Come First?

  • The first website was created in 1991 by Tim Berners-Lee
  • It was hosted on a NeXT computer
  • Purpose: To share information about the World Wide Web project
  • Website URL: info.cern.ch

Who Did It First and How?

  • Tim Berners-Lee invented the World Wide Web
  • Tools used: HTML (HyperText Markup Language)
  • First website contained links to other CERN-related pages
  • Technologies: HTTP protocol, web browsers, and servers
